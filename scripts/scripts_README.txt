binarize2.py --> binarizes all images in a folder then puts them in a sub-folder named "binarized"
confidence.py --> script to copy characters compiled in one folder into their individual folders.
create_sentence.py --> takes all the word level images in a folder then create a sentence by random
filter_dataset.py --> filters dataset in a specific amount per character (currently 50)
grayscale.py --> turns image to grayscale
ground_truth_word.py --> creates a ground truth text file for a word
gt.py --> creates a ground truth text file for all images in a folder
syllable_mapping.py --> creates a generalized ground truth for characters with multiple meanings (e.g. po/pu)
upscale.py --> upscales all the images in the folder (currently set to 128x128)

important to use:
binarize2.py
filter_dataset.py
gt.py
upscale.py
